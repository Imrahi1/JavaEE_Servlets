JSP - это файл с HTML кодом в который можно удобно вставлять Java код. Весь код из этого файла потом конвертируется в метод doGet

HTML
<% %> - Не выдает на экран (не обязаны вернуть строку класса String) - нужен для логики
<%= %> - Выдает на экран (обязаны вернуть строку класса String) - нужен для вывода

JSP
out.println - можно вывести что либо
<%@page import="путь к модулю" - позволяет импортировать модули, в том числе и свои.
